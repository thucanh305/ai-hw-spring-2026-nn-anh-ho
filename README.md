# ai-hw-spring-2026-nn-anh-ho
This project's idea is to build a small image recognition neural network using MNIST dataset. The goal is to:
- train models on the MNIST training set
- evaluate them on the MNIST test set.

It compares three model types:

- Shallow Multi-Layer Perceptron (MLP)
- Convolutional Neural Network (CNN)
- Transformer Encoder
